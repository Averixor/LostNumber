Lost Number — єдиний проєкт (консолідація 2026-07-01)
=====================================================

Це ОДИН канонічний репозиторій. Архіви good-project, handoff і «Нова тека»
вже злиті сюди. Деталі: MERGE_NOTES.md

Структура
---------
  godot/                   — Godot 4 → Android для Play Market (ОСНОВНИЙ)
    Boot.tscn → App.tscn → екрани (ScreenRouter autoload, back-stack)
  js/, css/, index.html     — web-еталон UI/логіки (PWA, Capacitor legacy)
  android/                 — Capacitor/WebView (legacy, не primary ship)
  assets/, public/audio/    — графіка, звук (джерело для godot/assets/ui/)
  store/                   — Play Console: іконки, описи, скріншоти
  build/godot/android/     — готові APK/AAB (якщо зібрані)
  docs/, godot/docs/       — документація (VISUAL_PORT_MAP.md — parity tracker)

Швидкий старт (браузер)
-----------------------
  npm ci
  npm run build:pages
  # відкрити _site/index.html або Live Server

Godot на телефон (рекомендовано)
--------------------------------
  npm run godot:import
  npm run godot:android:debug
  adb uninstall com.averixor.lostnumber.dev 2>/dev/null || true
  adb install -r build/godot/android/lost-number-debug.apk

  Запуск у редакторі: godot4 --path godot
  main_scene: Boot.tscn → App.tscn → MainMenu (ScreenRouter)
  Тести: npm run godot:test:all

Play Market (AAB)
-----------------
  npm run godot:android:release
  # завантажити build/godot/android/lost-number.aab у Play Console
  # package: com.averixor.lostnumber, version 2.1.4

Legacy Capacitor Android
------------------------
  npm ci && npm run android:debug:cheats
  # WebView — не для Play, якщо вже є Godot AAB

Чого немає в git (нормально)

  android/keystore*      — локальний підпис
  godot/android/         — генерується при export
  build/                 — збірки
