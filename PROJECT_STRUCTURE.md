# LostNumber — структура проекта

Проект не требует сборщика. Открывать нужно `index.html`.

## Основные папки

- `css/` — стили по блокам интерфейса.
- `js/bootstrap/` — окружение, запуск игры, глобальные обработчики.
- `js/core/` — базовая логика правил, событий, генератора и ядра.
- `js/system/` — хранение, звук, платформа, ошибки, i18n.
- `js/game/` — игровые системы: бонусы, колесо, задания, состояние.
- `js/game/grid/` — логика игрового поля, разбитая на физику, рендер, заморозку, safety-операции.
- `js/ui/` — экраны, меню, настройки, оверлеи.
- `js/app/` — главный класс `LostNumberGame` и методы, разложенные по смысловым блокам.

## Точка входа

`index.html` подключает все файлы обычными `<script>` без `type="module"`, чтобы игра запускалась локально без сервера и без сборки.
